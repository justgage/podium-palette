# Padding & Margin

Padding and Margin classes work like so

`.p{side?}-{size}` and `.m{side?}-{size}`

# Padding

And ones for each side (we've not shown all the sizes in this case).

- `pl-{size}` (left)
- `pt-{size}` (top)
- `pb-{size}` (bottom)
- `pr-{size}` (right)
