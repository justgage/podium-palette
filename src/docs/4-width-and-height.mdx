# Width & Height

Width and Height are set with `w-{size}` and `h-{size}`.

## Fixed Widths

## Max width

Max widths are very useful for responsive design. Combined with `w-full` they
provide useful utilities for making designs that stop growing at a certain width
but still allow themselves to be squished if needed.

<Playground>
  <div className="flex flex-col items-center">
    <div className="max-w-48 w-full bg-mystic m-4">.max-w-48</div>
    <div className="max-w-56 w-full bg-mystic m-4">.max-w-56</div>
    <div className="max-w-64 w-full bg-mystic m-4">.max-w-64</div>
    <div className="max-w-128 w-full bg-mystic m-4">.max-w-128</div>
  </div>
</Playground>
