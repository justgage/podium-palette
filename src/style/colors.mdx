# Colors

Podium CSS provides css classes to assign colors of Podium's branding.

There's two types, ones that start with `text-` and ones that start with `bg-`.

## Background Color

Backgrounds start with `bg-` such as `bg-blue`.

import settings from '../../tailwind.js';
console.log(settings);
const bg = Object.entries(settings.backgroundColors);
bg.sort(([k1, v1], [k2, v2]) => (v1 < v2 ? -1 : 1));
const textColor = Object.entries(settings.textColors);
textColor.sort(([k1, v1], [k2, v2]) => (v1 < v2 ? -1 : 1));
console.log(bg);

<div
  className="p-5 rounded-lg"
  style={{
    backgroundImage:
      'linear-gradient(45deg, lightgray 25%, transparent 25%), linear-gradient(-45deg, lightgray 25%, transparent 25%), linear-gradient(45deg, transparent 75%, lightgray 75%), linear-gradient(-45deg, transparent 75%, lightgray 75%)',
    backgroundSize: '20px 20px',
    backgroundPosition: '0 0, 0 10px, 10px -10px, -10px 0px'
  }}
>
  {bg.map(([k, v]) => (
    <div key={k} className={`p-3 bg-${k} ${v < '#adb6be' ? 'text-white' : ''}`}>
      bg-{k} ({v})
    </div>
  ))}
</div>

## Text Color

Backgrounds start with `text-` such as `text-blue`.

<div className="p-5">
  {textColor.map(([k, v]) => (
    <div

    >
      text-{k} ({v})
    </div>
  ))}
</div>
