# Buttons

## Primary

These should be the main call to action (CTA).
You can modify the size by using padding and text size.

<Playground>
  <button className="btn btn--primary px-5 py-4 mr-4">Primary Button</button>
  <button className="btn btn--primary px-4 py-3 mr-4">Primary Button</button>
  <button className="btn btn--primary px-3 py-2 text-sm mr-4">
    Primary Button
  </button>
</Playground>

## Secondary

These should be the secondary call to action, usually near a primary button.

<Playground>
  <button className="btn btn--secondary px-5 py-4 mr-4">Secondary Button</button>
  <button className="btn btn--secondary px-4 py-3 mr-4">Secondary Button</button>
  <button className="btn btn--secondary px-3 py-2 text-sm mr-4">
    Secondary Button
  </button>
</Playground>

## Text

Sometimes you want a button that doesn't have a visible border till you hover.

<Playground>
  <button className="btn text-blue hover:bg-whiteSmoke px-4 py-3 mr-4">
    Text Button
  </button>
  <button className="btn text-steel hover:bg-whiteSmoke px-4 py-3 mr-4">
    Text Button
  </button>
  <button className="btn text-red hover:bg-whiteSmoke px-4 py-3 mr-4">
    Text Button
  </button>
</Playground>

## Full Width

Just add `w-full` to make a button full width

<Playground>
  <button className="btn btn--primary px-5 py-4 mr-4 w-full">Primary Button</button>
</Playground>

## Disabled

`btn--disabled` will lighten the button with opacity and change the cursor.
Make sure not to use btn--primary or other color variants that set a hover color.

<Playground>
  <button disabled className="btn btn--disabled bg-blue text-white px-5 py-4 mr-4">
    Disabled Button
  </button>
</Playground>
